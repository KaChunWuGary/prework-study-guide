Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git merge: merges x branch to current branch
  • mkdir and touch creates folders and files
  • git clone SSH_key_link downloads repository
  • git add -A then git commit -m "description here" to commit changes, this updates current branch with the changes
  • To upload, double check local branch with remote repo branch with git pull origin main to make sure they match, then git push origin branch_name to push changes onto remote repo
  • next steps is to follow github making sure base branch is main, and compare is the branch with the changes
  • Create pull request -> peer review -> push/merge as needed
